This repository contains a collection of C programming solutions developed as part of my Master of Computer Applications (MCA) coursework for the subject Problem Solving.

🎯 About the Repository

The programs included here focus on building strong problem-solving skills using the C programming language. They cover fundamental concepts, logical thinking, and algorithmic approaches commonly taught in MCA curriculum.

🧠 Topics Covered

Basic C Programs (Input/Output, Operators)

Conditional Statements (if-else, switch)

Loops (for, while, do-while)

Functions and Recursion

Arrays and Strings

Pointers

Structures and Unions

File Handling

Searching and Sorting Algorithms

Mathematical and Logical Problems

💡 Purpose

To strengthen problem-solving ability

To practice structured programming in C

To prepare for lab exams and interviews

To build a strong foundation in programming concepts

🛠️ Technologies Used

Language: C

Compiler: GCC / Turbo C (depending on programs)

Platform: Windows / Linux
